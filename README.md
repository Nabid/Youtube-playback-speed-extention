# Youtube-playback-speed-extention
Youtube playback speed extention for Google Chrome

<li>Alt + 0 = Set speed to Normal (1)</li>
            <li>Alt + + = Increases playback speed by 0.25</li>
            <li>Alt + - = Decreases playback speed by 0.25</li>
